# Exercises: Classes and Instances

1. Complete BankAccount with deposit, guarded withdrawal, and displayed balance.
2. Create two accounts and prove their balances are independent.
3. Add a class-level currency and an alternate constructor from whole pesos.
4. Demonstrate a shared mutable class attribute, then repair it as instance state.
5. Test invalid owner, opening balance, deposit, and withdrawal.
6. Define `InsufficientStockError` with requested and available attributes, then raise and inspect it.
