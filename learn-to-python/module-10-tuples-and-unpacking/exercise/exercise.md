# Exercises: Tuples and Unpacking

1. Create a tuple containing a product name, price, and stock state. Unpack and print each value.
2. Create an empty tuple and a one-item tuple, then print their types and lengths.
3. Unpack `(1, 2, 3, 4, 5)` into first, middle, and last.
4. Swap two string values with one assignment.
5. Put a list inside a tuple, mutate the list, and explain why the tuple rule was not broken.
