# Exercises: Numbers and Arithmetic

1. Calculate subtotal from unit price and quantity, then add an 8 percent tax.
2. Split 29 items into boxes of 6. Print full boxes and remaining items.
3. Predict and run `-7 // 3` and `-7 % 3`; verify `divisor * quotient + remainder` reconstructs the original.
4. Compare `0.1 + 0.2` with `0.3` and explain the result.
