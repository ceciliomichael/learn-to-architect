# Exercises: Text and String Operations

1. Store `  Python Course  `, strip it, and print original and cleaned values with visible brackets.
2. Print the first, last, and middle six characters of `TypeScript` using indexes and slices.
3. Replace the commas in `red,green,blue` with ` | `.
4. Use an f-string to display an item, quantity, unit price, and two-decimal subtotal.
