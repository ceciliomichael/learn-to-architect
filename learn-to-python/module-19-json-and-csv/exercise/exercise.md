# Exercises: Work with JSON and CSV Data

1. Write and read a UTF-8 JSON list of book dictionaries.
2. Validate that the result is a list and every item has string title and nonnegative numeric price.
3. Write three records through `csv.DictWriter`, then read and convert numeric fields.
4. Reject a CSV whose required header is missing.
5. Explain why a successful parse does not prove trusted content.
