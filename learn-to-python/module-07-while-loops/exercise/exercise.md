# Exercises: Repeat with While Loops

1. Print 5 down through 1, then `Go`.
2. Keep asking until the user enters `yes` or `no`.
3. Ask for a nonnegative whole number using `isdecimal`, then convert it.
4. Give three attempts to enter a fixed code and use loop `else` for failure.
5. Create an infinite loop accidentally in a disposable file, interrupt it, and repair its progress step.
