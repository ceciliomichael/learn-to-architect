# Exercises: Modules, Imports, and Packages

1. Move two calculation functions into `calculations.py` and import them from `app.py`.
2. Put all interactive behavior in `main` behind a main guard.
3. Import `app` from another file and prove the interactive behavior does not run.
4. Create `learning_tools` with `__init__.py`, `text.py`, and `numbers.py`.
5. Run one module with `python -m` from the project root.
