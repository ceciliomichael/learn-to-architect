# Exercise Solutions: Numbers and Arithmetic

```python
unit_price = 12.50
quantity = 3
subtotal = unit_price * quantity
total = subtotal * 1.08
print(subtotal, total)

items = 29
box_size = 6
print(items // box_size)
print(items % box_size)

quotient = -7 // 3
remainder = -7 % 3
print(quotient, remainder)
print(3 * quotient + remainder)

print(0.1 + 0.2)
print(0.1 + 0.2 == 0.3)
```

There are 4 full boxes and 5 remaining items. Negative floor division gives -3 with remainder 2. The float equality is false because the binary approximations differ.
