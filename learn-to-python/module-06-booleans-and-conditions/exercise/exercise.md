# Exercises: Booleans and Conditions

1. Classify an entered score as 90 or higher, 75 to 89, 60 to 74, or below 60.
2. Allow access only when a user is active and their role is `owner` or `editor`.
3. Ask for trimmed text and reject an empty answer.
4. Safely divide only when the denominator is not zero.
5. Explain why branch order matters in the score program.
