# Exercises: Threads, Processes, and Queues

1. Read sizes of disposable files with a thread pool and observe every future.
2. Create a deliberate shared-counter race, then repair it with one owner or a lock.
3. Use a Queue with producer, consumer, sentinel, task_done, and join.
4. Calculate independent CPU values in a process pool behind a main guard.
5. Measure sequential and concurrent versions before stating which is faster.
