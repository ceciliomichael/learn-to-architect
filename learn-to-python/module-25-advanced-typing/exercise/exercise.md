# Exercises: Protocols, Generics, and Typed Data

1. Write generic `last` that rejects an empty list.
2. Define a protocol for objects with `save(text: str) -> None` and a function that uses it.
3. Define a TypedDict for a product row with optional description using `NotRequired`.
4. Define `OrderId` with NewType and an order status Literal.
5. Explain why none of these replace JSON validation.
