# Exercises: Define and Call Functions

1. Write `area(width, height)` that returns a rectangle area.
2. Write `is_even(number)` that returns a boolean.
3. Write `format_price(amount)` that returns two-decimal display text without printing.
4. Compose subtotal and tax functions for an order.
5. Call a function with no explicit return and inspect its result.
6. Return the smallest and largest list values as one tuple and unpack the result.
