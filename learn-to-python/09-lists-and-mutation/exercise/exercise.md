# Exercises: Lists and Mutation

1. Create a task list, append two tasks, insert one first, change one, and pop the completed task.
2. Assign the list to a second name, mutate through it, and explain both outputs.
3. Make a shallow copy and prove an outer append is independent.
4. Compare `sorted(values)` with `values.sort()` and their return values.
5. Build a new list containing only positive numbers with an ordinary loop.
6. Split `red,green,blue` on commas and join the parts with ` | `.
