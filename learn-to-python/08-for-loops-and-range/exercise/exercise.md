# Exercises: Repeat with For and Range

1. Print integers 1 through 10.
2. Print even integers from 10 down to 2.
3. Print numbered characters from a word using `enumerate` starting at 1.
4. Pair the characters in `ABC` with integers 1 through 3 using strict zip.
5. Sum integers 1 through 100 with a loop and compare with `sum(range(1, 101))`.
