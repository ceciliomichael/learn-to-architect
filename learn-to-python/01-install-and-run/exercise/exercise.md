# Exercises: Install Python and Run a Program

1. Display the installed Python version.
2. Use the interactive prompt to print your first name and the number 3.
3. Create `greeting.py` that prints three separate lines.
4. Run it from its folder.
5. Add a reference to `unknown_value`, run it, and record the exception type and source line.
6. Remove the error and prove the program works again.
