# Exercise Solutions: Install Python and Run a Program

```text
python --version
python
```

At the prompt:

```python
print("Ana")
print(3)
exit()
```

One valid `greeting.py` is:

```python
print("Good morning.")
print("I am learning Python.")
print("This program has three lines of output.")
```

Run `python greeting.py`. Adding `print(unknown_value)` produces `NameError` on that line because no value has been assigned to the name. Remove it and rerun.
