# Exercises: Names, Scope, Objects, and Mutation

1. Read a global constant inside a function, then refactor it into a parameter.
2. Pass a list to one function that mutates it and one that rebinds its local parameter. Compare caller results.
3. Make equal but distinct lists and compare `==` with `is`.
4. Demonstrate shallow-copy sharing with a nested list.
5. Repair an `UnboundLocalError` by returning a calculated value instead of using global state.
