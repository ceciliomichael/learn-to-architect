# Exercises: Input and Conversion

1. Ask for a name and print a greeting.
2. Ask for two whole numbers, convert them, and print their sum.
3. Ask for quantity and price, then display a two-decimal subtotal.
4. Enter invalid whole-number text once, read `ValueError`, then rerun with valid input.
5. Demonstrate the difference between `"3" + "4"` and `int("3") + int("4")`.
