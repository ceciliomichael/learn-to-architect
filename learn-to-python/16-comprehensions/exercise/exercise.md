# Exercises: Comprehensions and Generator Expressions

1. Convert a loop that uppercases nonempty trimmed names into a list comprehension.
2. Build a set of word lengths.
3. Build a dictionary mapping integers 1 through 5 to their cubes.
4. Create even or odd labels for integers 0 through 9.
5. Sum squares through a generator expression and prove a stored generator is exhausted after one pass.
