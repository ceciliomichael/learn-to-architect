# Exercises: Dictionaries

1. Create a student dictionary with ID, name, and score. Add active state and update the score.
2. Read one required key with brackets and one optional key with `get` and a fallback.
3. Print every key-value pair with `items`.
4. Merge default settings with user settings and prove the right side wins.
5. Shallow-copy a dictionary containing a list, mutate the nested list through the copy, and explain both results.
