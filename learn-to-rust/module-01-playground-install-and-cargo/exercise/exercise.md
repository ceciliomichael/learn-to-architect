# Exercises: Use the Rust Playground and Install Rust

1. Run the first program in the official Playground.
2. Change it to print three lines.
3. Run Rustfmt and observe the layout.
4. Introduce one syntax error, read it, and repair it.
5. If Rust is local, create a Rust 2024 Cargo project and run `cargo check`, `cargo run`, and `cargo fmt --check`.
