# Exercises: Branches, HEAD, and Switching

## Exercise 1: Create a clean practice repository

1. Inside `git-course-practice`, create `branch-practice` and initialize it.
2. Create `story.txt` with `Beginning.`
3. Stage and commit it with `Start story`.
4. Confirm the current branch is `main`.

## Exercise 2: Move only one branch

1. Create and switch to `add-middle`.
2. Add `Middle.` to the file and commit it.
3. Draw where `main`, `add-middle`, and `HEAD` point.
4. Switch to `main` and inspect the file.
5. Explain why `Middle.` is absent.

## Exercise 3: Inspect and rename

1. Show all branches and the decorated graph.
2. Switch to `add-middle`.
3. Rename it to `story-middle`.
4. Confirm the new name and clean status.
