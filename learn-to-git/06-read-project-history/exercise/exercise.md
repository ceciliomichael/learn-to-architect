# Exercises: Read Project History

## Exercise 1: Read the whole story

1. Show the full log.
2. Show a compact decorated log.
3. Identify the newest and oldest commit subjects.
4. Copy the short hash of the oldest commit.

## Exercise 2: Inspect exact evidence

1. Show the oldest commit using its hash.
2. Show only its file statistics.
3. Show `HEAD` and `HEAD^` in compact form.
4. Explain the relationship between them.

## Exercise 3: Follow and search

1. Show commits affecting `notes.txt` or its current renamed path.
2. Search messages for a word you used in an earlier commit subject.
3. Run the graph view for all references.
