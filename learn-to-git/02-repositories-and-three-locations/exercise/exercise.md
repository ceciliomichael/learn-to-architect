# Exercises: Repositories and Git's Three Locations

Work inside `git-course-practice`.

## Exercise 1: Initialize safely

1. Create and enter `three-locations`.
2. Print the current folder.
3. Initialize a repository.
4. Ask Git for the repository root.
5. Run status.

## Exercise 2: Observe an untracked file

1. Create `notes.txt` containing `First working tree version.`
2. Run status.
3. Identify the word Git uses for the new file.

## Exercise 3: Prove staging is a copy

1. Stage `notes.txt`.
2. Add `Second working tree version.` as another line without staging again.
3. Run status. Confirm that `notes.txt` appears in two sections.
4. Stage the file again.
5. Run status and describe what changed.

Keep this repository for Module 03.
