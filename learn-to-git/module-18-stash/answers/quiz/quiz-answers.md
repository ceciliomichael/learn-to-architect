# Quiz Answers: Set Work Aside with Stash

1. Staged and unstaged changes to tracked files.
2. `-u`, also written `--include-untracked`.
3. Apply keeps the stash entry. Pop tries to apply it and then remove it after a successful application.
4. The current snapshot may have changed the same content since the stash was created.
5. When you want to resume the work near its original base and keep it on a real branch.
