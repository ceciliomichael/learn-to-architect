# Exercises: Install Git and Create a Safe Practice Area

Complete these steps in a terminal.

## Exercise 1: Check your tools

1. Print the current folder.
2. Print the installed Git version.
3. Print the configured global user name and email.
4. If either identity value is missing, configure it with your own value.

## Exercise 2: Create the course folder

1. Move to a location where you keep practice work.
2. Create a folder named `git-course-practice`.
3. Enter the folder.
4. Print the current folder and confirm its final name.

## Exercise 3: Set and verify the default branch

1. Configure `main` as the default initial branch.
2. Read that setting back from Git.

Do not initialize a repository yet. That is the next module's job.
