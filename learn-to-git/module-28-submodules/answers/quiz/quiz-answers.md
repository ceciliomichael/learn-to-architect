# Quiz Answers: Manage Nested Repositories with Submodules

1. The exact selected commit identifier through a gitlink entry.
2. `.gitmodules`.
3. Reproduction needs the exact recorded commit, not necessarily the tip of a child branch.
4. The child repository gets the content commit, and the parent gets a commit updating the gitlink.
5. Other users must be able to fetch the child object named by the parent.
