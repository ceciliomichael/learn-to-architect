## Your Tasks

Write a generic function `getFirstElement<T>(arr: T[]): T | undefined` that:
- Returns `arr[0]` if the array has at least one item.
- Returns `undefined` if the array is empty.

1. A `string[]`  -  log the result and write a comment showing what TypeScript infers as the return type.
2. A `number[]`  -  log the result and write a comment showing what TypeScript infers.
3. An empty array `[]`  -  for this call, you will need to specify the type explicitly as `getFirstElement<string>([])`. Log the result.
