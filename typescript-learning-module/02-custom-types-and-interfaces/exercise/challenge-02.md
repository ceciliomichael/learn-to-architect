# Challenge 02: Extend Blueprints Two Ways

TypeScript lets you build on existing interfaces in two different ways.

## Your Tasks

**Part 1 - Using `interface` and `extends`:**
1. Create an `interface Animal` with `name: string` and `age: number`.
2. Create an `interface Bird` that `extends Animal` and adds `wingspan: number` and `canFly: boolean`.
3. Create an object `parrot` typed as `Bird` with all four properties filled in.

**Part 2 - Using `type` aliases and `&` (intersection):**
1. Create a `type TypeAnimal` with `name: string` and `age: number`.
2. Create a `type TypeBird` by intersecting `TypeAnimal` with `{ wingspan: number; canFly: boolean }`.
3. Create an object `eagle` typed as `TypeBird` with all four properties filled in.
