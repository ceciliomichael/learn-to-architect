# Exercise: Structures and Data Ownership

## Goal

Apply the module's rules in a complete program that you can explain line by line.

## Steps

1. Define a struct Rectangle with double width and height fields.
2. Write a static area function that accepts a pointer to a const Rectangle.
3. Return 0.0 for NULL and width * height otherwise.
4. Create a rectangle with designated initializers.
5. Print its area with one digit after the decimal point.

## Success check

The fields form one value, the function does not modify it, null is handled as documented, and the area is 44.0.

Use the same strict C17 command from the lesson unless a step says otherwise. Attempt the work before opening the solution.
