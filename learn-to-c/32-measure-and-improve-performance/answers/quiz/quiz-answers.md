# Quiz Answers: Measure and Improve Performance

1. **Why use representative input?**

   Different sizes and shapes can exercise different costs.

2. **Why print or otherwise consume the result?**

   To stop the optimizer from removing unobservable work.

3. **Does one timing prove improvement?**

   No. Measurements vary and require repeated comparable trials.

4. **What follows an optimization?**

   Complete correctness tests and controlled remeasurement.

5. **Where should optimization begin?**

   At measured hotspots, considering algorithms and data movement first.
