# Quiz Answers: Portability, Standards, and C23

1. **Does a newer compiler automatically select C23?**

   No.

2. **What value identifies C17?**

   201710L.

3. **What value identifies C23?**

   202311L.

4. **Why can migration require more than a compiler?**

   Libraries, analyzers, dependencies, and all supported platforms need adequate support.

5. **How should platform code be handled?**

   Label it, document it, and isolate it behind a small boundary.
