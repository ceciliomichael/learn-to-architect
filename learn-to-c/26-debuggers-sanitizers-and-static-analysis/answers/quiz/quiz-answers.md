# Quiz Answers: Debuggers, Sanitizers, and Static Analysis

1. **What does a debugger explain?**

   The current state and flow of one selected execution.

2. **What does a sanitizer add?**

   Runtime checks for supported defect classes.

3. **Does a clean sanitizer run prove safety?**

   No. Other paths and unsupported defect classes remain.

4. **What can static analysis inspect?**

   Potential control and data paths without running them all.

5. **Why understand a warning first?**

   Blind suppression can hide the cause or weaken a correct contract.
