# Exercise: Testing and Assertions

## Steps

1. Write absolute_value for values greater than INT_MIN.
2. Use assert to state that precondition.
3. Test a negative value, zero, and a positive value.
4. Print tests passed only after every assertion.
5. Run the strict build and an optimized build.

## Success check

Three behavior classes are checked, INT_MIN is an explicit programmer precondition, and success prints only after all checks.
