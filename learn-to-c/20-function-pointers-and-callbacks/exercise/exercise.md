# Exercise: Function Pointers and Callbacks

## Goal

Apply the module's rules in a complete program that you can explain line by line.

## Steps

1. Create an array containing 5, 1, 8, and 3.
2. Write a qsort comparator for descending int order.
3. Use relational comparisons rather than subtraction.
4. Sort and print all four elements.
5. Explain the meaning of negative, zero, and positive callback results.

## Success check

The callback exactly matches qsort's type, avoids overflowing subtraction, and produces 8 5 3 1.

Use the same strict C17 command from the lesson unless a step says otherwise. Attempt the work before opening the solution.
