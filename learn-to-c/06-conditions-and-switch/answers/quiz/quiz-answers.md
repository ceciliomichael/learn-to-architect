# Quiz Answers: Conditions and Switch

1. **What numeric value is false in a C condition?**

   Zero is false; nonzero scalar values are true.

2. **Why does order matter in an else-if chain?**

   Only the first true branch runs, so an earlier broad condition can hide a later narrow one.

3. **What happens when break is omitted from a switch case?**

   Execution continues into following cases unless another control transfer stops it.

4. **When is switch a good fit?**

   It is a good fit for choices among exact integral, character, or enumeration values.

5. **What is the difference between = and ==?**

   The first assigns; the second compares for equality.
