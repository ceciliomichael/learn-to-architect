# Exercise: Variables, Types, Constants, and Limits

## Goal

Apply the module's rules in a complete program that you can explain line by line.

## Steps

1. Create const variables for a room number, an item count, and a temperature in degrees Celsius.
2. Use int for the room, size_t for the count, and double for the temperature.
3. Print each value with a label and the matching format specifier.
4. Also print INT_MIN from limits.h.
5. Compile with the full warning set and repair every warning.

## Success check

All values are initialized, every format matches its argument, and the strict build has no warnings.

Use the same strict C17 command from the lesson unless a step says otherwise. Attempt the work before opening the solution.
