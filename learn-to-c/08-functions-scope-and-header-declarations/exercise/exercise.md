# Exercise: Functions, Scope, and Header Declarations

## Goal

Apply the module's rules in a complete program that you can explain line by line.

## Steps

1. Declare a static function named square before main.
2. Give it one int parameter and an int return type.
3. Define it after main and return value * value.
4. Call it for 7 and print the result.
5. Also call it for -3 and explain why the result is positive.

## Success check

The declaration is visible before both calls, the definition matches it exactly, and the output is 49 followed by 9.

Use the same strict C17 command from the lesson unless a step says otherwise. Attempt the work before opening the solution.
