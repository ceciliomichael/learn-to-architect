# Exercise: Conditions and Switch

## Goal

Apply the module's rules in a complete program that you can explain line by line.

## Steps

1. Create an int named month with a value from 1 through 12.
2. Use if to reject values outside that range.
3. For a valid value, use switch to print First quarter for 1, 2, or 3 and Later quarter for the other valid months.
4. Ensure every switch path ends deliberately.
5. Test 0, 2, and 12.

## Success check

Values outside 1 through 12 are rejected, 1 through 3 select the first quarter, and all other valid values select the later-quarter path.

Use the same strict C17 command from the lesson unless a step says otherwise. Attempt the work before opening the solution.
