# Exercise: Online Compiler and Local Toolchain Choices

## Goal

Apply the module's rules in a complete program that you can explain line by line.

## Steps

1. Run the program in the browser compiler.
2. Change the message to Learning one step at a time.
3. Delete the semicolon after puts, compile, read the first error, and repair it.
4. If GCC or Clang is installed, save the file as main.c and run the strict compile command.
5. Run the produced program and confirm the browser and local output agree.

## Success check

The program compiles without warnings and prints exactly one line.

Use the same strict C17 command from the lesson unless a step says otherwise. Attempt the work before opening the solution.
