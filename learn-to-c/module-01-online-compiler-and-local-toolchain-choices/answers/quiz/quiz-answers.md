# Quiz Answers: Online Compiler and Local Toolchain Choices

1. **What does the compiler do?**

   It checks C source and translates valid source into object code that can become an executable.

2. **Why use explicit warning flags?**

   They expose suspicious conversions and other mistakes that a basic compile may accept silently.

3. **When is a browser compiler enough?**

   It is enough for small, non-private, single-file exercises that do not require unrestricted files, libraries, networks, or diagnostic tools.

4. **Why fix the first relevant error first?**

   A single early syntax error can cause several later messages that disappear after the first problem is fixed.

5. **What does main(void) communicate?**

   It declares the program entry function and says it receives no parameters.
