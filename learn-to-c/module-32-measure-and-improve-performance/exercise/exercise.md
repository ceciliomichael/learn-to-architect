# Exercise: Measure and Improve Performance

## Steps

1. Run several trials with unchanged input and flags.
2. Verify the total every time.
3. Record every timing, not only the best.
4. Double the count and verify the new expected total.
5. Use profiler or generated-code evidence before naming a hotspot.

## Success check

The result remains observable at 49999995000000, environment and flags are recorded, and timings are treated as noisy observations.
