# Quiz Answers: Security Hardening

1. **Which inputs are untrusted?**

   All values crossing a process or trust boundary.

2. **Why is truncation risky?**

   The validated prefix can differ from the intended value.

3. **Do stack protections make bad bounds correct?**

   No.

4. **What accompanies a buffer?**

   Capacity and verified valid length.

5. **Why use several defense layers?**

   Each detects or limits different failures and none is complete.
