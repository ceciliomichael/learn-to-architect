# Exercise: Dynamic Memory and Ownership

## Goal

Apply the module's rules in a complete program that you can explain line by line.

## Steps

1. Allocate an array for four double values.
2. Check the count multiplication before malloc and check the returned pointer.
3. Store 1.5, 2.5, 3.5, and 4.5.
4. Sum and print the values.
5. Free the allocation on every path after it succeeds.

## Success check

The requested byte count is proven representable, allocation failure is handled, ownership is released once, and the total is 12.0.

Use the same strict C17 command from the lesson unless a step says otherwise. Attempt the work before opening the solution.
