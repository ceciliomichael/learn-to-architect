# Exercise: Operators, Conversions, and Overflow

## Goal

Apply the module's rules in a complete program that you can explain line by line.

## Steps

1. Store the three int values 1200000000, 900000000, and 600000000.
2. Convert each operand to int64_t before adding it.
3. Print the total with PRId64.
4. Compute a double average by dividing the total by 3.0.
5. Explain in one sentence why total / 3 would have a different type.

## Success check

The total is 2700000000, the average is 900000000.0, and no overflowing int addition occurs first.

Use the same strict C17 command from the lesson unless a step says otherwise. Attempt the work before opening the solution.
