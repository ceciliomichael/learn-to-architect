# Exercise: Arrays, Pointer Arithmetic, and Bounds

## Goal

Apply the module's rules in a complete program that you can explain line by line.

## Steps

1. Create an array containing 2, 4, 6, and 8.
2. Write a static count_even function that accepts begin and end pointers.
3. Move a current pointer from begin up to but not including end.
4. Count elements whose remainder after division by 2 is zero.
5. Print the count.

## Success check

The loop stops at the one-past boundary without dereferencing it and prints 4.

Use the same strict C17 command from the lesson unless a step says otherwise. Attempt the work before opening the solution.
