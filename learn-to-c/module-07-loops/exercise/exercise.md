# Exercise: Loops

## Goal

Apply the module's rules in a complete program that you can explain line by line.

## Steps

1. Use a for loop to print the integers 1 through 5.
2. Use a while loop to count down from 3 through 1.
3. After the countdown, print Go.
4. Keep every loop condition finite and explain which update makes it stop.
5. Compile with conversion warnings enabled.

## Success check

The output contains 1 through 5, then 3 through 1, then Go, and both loops visibly move toward their stopping condition.

Use the same strict C17 command from the lesson unless a step says otherwise. Attempt the work before opening the solution.
