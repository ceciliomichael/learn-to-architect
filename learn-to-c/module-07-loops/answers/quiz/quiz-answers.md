# Quiz Answers: Loops

1. **What range does index < count describe when index starts at zero?**

   It describes indexes zero through count minus one.

2. **When does a do loop test its condition?**

   After running the loop body, so the body runs at least once.

3. **What should every finite loop make clear?**

   Its stopping condition and the update or event that eventually makes that condition false.

4. **What does continue do?**

   It skips the rest of the current iteration and proceeds to the next condition check or update.

5. **Why are off-by-one errors dangerous in C?**

   They can process the wrong number of items and can access memory outside an array.
