# Exercise: Arrays and Length Tracking

## Goal

Apply the module's rules in a complete program that you can explain line by line.

## Steps

1. Create an array containing 4, 7, 1, 9, and 3.
2. Compute its count in main.
3. Write a static maximum function that accepts the array and count.
4. Return the largest element. The exercise contract guarantees count is at least one.
5. Print the count and maximum.

## Success check

The function never uses an index equal to count, and the output reports 5 elements with a maximum of 9.

Use the same strict C17 command from the lesson unless a step says otherwise. Attempt the work before opening the solution.
