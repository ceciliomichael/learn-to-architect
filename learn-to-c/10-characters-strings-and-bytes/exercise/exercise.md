# Exercise: Characters, Strings, and Bytes

## Goal

Apply the module's rules in a complete program that you can explain line by line.

## Steps

1. Store the strings red and blue in separate const arrays.
2. Create a 24-byte destination.
3. Use snprintf to produce red + blue.
4. Reject an encoding error or truncation before using the result.
5. Print the combined string and its strlen value.

## Success check

The program proves the formatted result fits before treating it as a complete string and reports a length of 10.

Use the same strict C17 command from the lesson unless a step says otherwise. Attempt the work before opening the solution.
