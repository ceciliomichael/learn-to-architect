# Quiz answers

1. A dependency is identified by group, artifact, and version coordinates.
2. A scope or configuration states whether code is needed for compilation, tests, runtime, or another phase.
3. One direct dependency can bring many transitive dependencies that also require review.
4. A committed Maven or Gradle wrapper pins the build-tool distribution for the project.
5. Review source, maintainers, license, advisories, checksums, build plugins, and resolved changes before approval.
