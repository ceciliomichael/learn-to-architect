# Exercise: Console Input, Parsing, and Validation

1. Recreate the complete program from the lesson in a disposable folder.
2. Compile it with `javac --release 21 -Xlint:all Main.java`.
3. Confirm the expected behavior: a prompt and either accepted input or a correction message.
4. Test blank text, letters, 0, 1, 100, and 101.
5. Explain how each of the five lesson rules appears in the final program.

The solution must compile without warnings and must not remove validation merely to make an input pass.
