# Quiz answers

1. The collector may reclaim an object only after it is no longer strongly reachable.
2. System.gc is only a request and cleanup logic must not depend on collection timing.
3. Garbage collection does not replace close for operating-system and native resources.
4. A static collection, cache, listener, or ThreadLocal can retain objects unintentionally.
5. Use heap evidence and allocation profiles rather than guessing from source alone.
