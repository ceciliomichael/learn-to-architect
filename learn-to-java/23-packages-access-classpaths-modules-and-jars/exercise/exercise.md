# Exercise: Packages, Access, Classpaths, Modules, and JARs

1. Recreate the example in a disposable project using the documented source layout.
2. Run or verify it with javac --release 21 -Xlint:all Main.java, then java Main.
3. Confirm this behavior: named module: false and an empty package name.
4. Move Main into package course.app, compile with -d out, and create a runnable JAR with an explicit main class.
5. Explain how every listed contract is enforced and what would break if it were removed.

Do not commit generated caches, downloaded libraries, credentials, or real user data.
