# Quiz answers

1. The official Playground is useful for small public examples but not local files or installed libraries.
2. A JDK includes the runtime and development tools such as javac, jar, javadoc, and jshell.
3. Required course code targets Java 21, independent of the newest feature release.
4. A newer compiler uses --release 21 to check both source features and Java SE APIs.
5. Read the first compiler diagnostic and source line before editing.
