# Exercise: Source Files, Compilation, Class Files, and Main

1. Recreate the complete program from the lesson in a disposable folder.
2. Compile it with `javac --release 21 -Xlint:all Main.java`.
3. Confirm the expected behavior: three lines: source, compile, run.
4. Compile into an out directory with javac --release 21 -d out Main.java and run with java -cp out Main.

The solution must compile without warnings and must not remove validation merely to make an input pass.
