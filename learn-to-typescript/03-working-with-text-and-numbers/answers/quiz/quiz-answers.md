# Module 03 Quiz Answers

## Answer 1

```text
Ari has 15 points.
```

The expression inside the placeholder is calculated before its result is inserted into the string.

## Answer 2

`word.length` reads a property. It has no parentheses. `word.toUpperCase()` and `word.toLowerCase()` call methods and therefore use parentheses.

## Answer 3

```text
14
20
```

Multiplication normally happens before addition. Parentheses make the addition happen first in the second expression.

## Answer 4

It returns a string. The result is formatted text with a fixed number of decimal places, which is useful for display. It should not be treated as a number unless the program deliberately converts and validates it.

## Answer 5

```typescript
const price = 50;
const quantity = 3;
const total = price * quantity;
```

The price must be numeric, and the total for equal-price items is price multiplied by quantity.
