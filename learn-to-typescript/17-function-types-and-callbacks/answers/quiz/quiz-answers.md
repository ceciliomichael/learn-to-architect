# Module 17 Quiz Answers

## Answer 1

`callback` refers to the function value so it can be passed or stored. `callback()` calls it immediately.

## Answer 2

It is a function that accepts a `Task` value and returns a boolean.

## Answer 3

It is `string | undefined` because the caller may omit it.

## Answer 4

The default value is used when the argument is omitted or undefined, so the body receives the inferred non-undefined type of that default.

## Answer 5

The caller promises not to use a returned value. It treats the callback as an action.
