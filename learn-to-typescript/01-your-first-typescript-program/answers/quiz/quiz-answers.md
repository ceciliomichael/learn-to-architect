# Module 01 Quiz Answers

## Answer 1

```text
One
Three
```

The second statement begins with `//`, so it is a comment and does not run.

## Answer 2

Source code is the text a programmer writes, including language punctuation such as parentheses and quotation marks. Output is the information produced when the program runs. Output does not automatically show all the punctuation from the source.

## Answer 3

**B** is valid:

```typescript
console.log("Hello");
```

Letter case matters, text needs matching quotation marks, and a bare name must have been declared before it can be used.

## Answer 4

`//` tells TypeScript that the rest of that line is a comment for people reading the source. Comments are ignored when the program runs.

## Answer 5

```typescript
console.log("Ready");
```

The call needed a closing parenthesis. The semicolon consistently marks the end of the statement.
