# Module 01 Exercises

Complete these tasks in a file named `index.ts`.

## Warm-up: Change a working program

Start with:

```typescript
console.log("Good morning");
```

1. Change the message to greet yourself by name.
2. Add a second statement that prints `I ran my first program.`
3. Run the file and confirm that both lines appear in order.

## Guided exercise: A three-step recipe

Write a program that prints these instructions, one per line:

```text
1. Put bread on a plate.
2. Add the filling.
3. Close the sandwich.
```

Follow these steps:

1. Write one `console.log` statement for the first instruction.
2. Run the file before adding more code.
3. Add the second and third statements.
4. Add a comment above the statements that says what the program prints.
5. Run the finished file.

## Independent exercise: A short introduction

Write a program that prints exactly four lines:

1. Your name
2. A place you would like to visit
3. Something you would like to build with code
4. `End of introduction.`

Requirements:

- Use four separate `console.log` statements.
- Include one single-line comment.
- Do not print quotation marks around the messages.

## Debugging task

The following program has three problems. Correct it so it prints both messages.

```typescript
Console.log("Starting now");
console.log("Practice makes progress);
console.log("Finished")
```
