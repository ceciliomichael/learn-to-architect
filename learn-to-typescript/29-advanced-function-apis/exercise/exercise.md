# Module 29 Exercises

## Exercise 1: Rest values

Write `joinWith(separator: string, ...values: string[]): string`. Test zero, one, and several values.

## Exercise 2: Tuple spread

Write `createRange(start: number, end: number): number[]`. Store arguments in a tuple and spread them into the call.

## Exercise 3: Union or overload

Write a function that accepts a string or number and always returns a string. Use a union. Explain why overloads add no useful relationship.

## Exercise 4: Related overloads

Overload `double` so a number returns a number and a string returns a string repeated twice. Implement one body with narrowing.

## Exercise 5: Generic mapper

Implement `mapValues` from the lesson and use it to turn numbers into labels. Confirm the returned type is string array.

## Completion checklist

- [ ] I used rest and spread with accurate types.
- [ ] I chose a union for a shared result.
- [ ] I used overloads only for a real call relationship.
- [ ] I connected callback output to result output.
