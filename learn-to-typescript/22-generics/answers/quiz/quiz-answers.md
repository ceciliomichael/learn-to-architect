# Module 22 Quiz Answers

## Answer 1

The returned value has the same chosen type as the input value for that call.

## Answer 2

The input array may be empty, so no first value exists.

## Answer 3

The empty array supplies no item from which TypeScript can infer the intended string type.

## Answer 4

The chosen type must have an `id` property whose value is a string.

## Answer 5

There is no relationship to preserve across positions. A concrete type or `unknown` is often clearer.
