# Learn React Step by Step

This specialization follows [Frontend Development with TypeScript](../learn-to-frontend-development/README.md). It assumes the shared HTML, CSS, TypeScript, DOM, HTTP, browser-security, testing, accessibility, and performance foundations instead of reteaching them through framework shortcuts.

## Local course project

Module 01 creates one Vite React and TypeScript project named `react-course`. Continue in that project instead of creating an unexplained new scaffold for every module. The course does not use deprecated Create React App instructions. Additional testing packages are installed when their module teaches them.
