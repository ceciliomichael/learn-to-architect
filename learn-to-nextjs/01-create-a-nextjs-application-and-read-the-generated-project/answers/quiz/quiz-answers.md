# Quiz Answers: Create a Next.js Application and Read the Generated Project

1. It downloads and runs the current official Next.js generator, creates a folder named `nextjs-course`, and configures npm as the package manager.
2. Choosing App Router makes folders and special files inside `app` define the routes.
3. `app/page.tsx` supplies the content for `/`. The root `app/layout.tsx` wraps pages and owns the required `html` and `body` elements plus shared metadata.
4. They are generated from source and dependencies. Manual edits are disposable, make the project hard to reproduce, and will be overwritten.
5. A production build checks that the full application compiles and that Next.js can prepare its routes and server output for deployment.
