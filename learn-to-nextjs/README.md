# Learn Next.js Step by Step

This specialization follows [React](../learn-to-react/README.md). It assumes the shared HTML, CSS, TypeScript, DOM, HTTP, browser-security, testing, accessibility, and performance foundations instead of reteaching them through framework shortcuts.

## Local course project

Module 01 creates one TypeScript App Router project named `nextjs-course` with the official `create-next-app` generator. Continue in that project. This course does not use Pages Router files. Database, runtime-validation, component-testing, and browser-testing packages are installed in the modules that first teach them.
