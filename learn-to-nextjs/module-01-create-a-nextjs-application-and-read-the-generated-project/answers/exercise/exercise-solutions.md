## Why this works

Create a current App Router project with the official project generator, then read its scripts, dependency versions, TypeScript settings, and app entry files before changing them. The reference keeps the example intentionally small so the lesson boundary remains visible. A larger solution is correct only when it preserves the same ownership, accessibility, runtime validation, and recovery rules.
