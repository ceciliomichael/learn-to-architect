# Learn Node.js and TypeScript Backend Development Step by Step

This course follows [Backend Foundations](../learn-to-backend-foundations/README.md), [TypeScript](../learn-to-typescript/README.md), and [SQL](../learn-to-sql/README.md). Complete the modules in order because later work assumes the contracts, failure models, and operational habits established earlier.

Every module contains a lesson, exercise, five-question quiz, complete reference solution, and explained quiz answers. Use only disposable data and local or dedicated practice services.

## TypeScript-first course project

Module 01 creates one strict TypeScript project named `node-api-course`. Source remains in `src`, compiled JavaScript goes to `dist`, and every dependency is installed locally and recorded in the lock file. Development execution never replaces `npm run typecheck`. Later modules must provide the installation command before importing a new package.
