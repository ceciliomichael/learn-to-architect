# Learn Go Backend Development Step by Step

This course follows [Backend Foundations](../learn-to-backend-foundations/README.md), [Go](../learn-to-go/README.md), and [SQL](../learn-to-sql/README.md). Complete the modules in order because later work assumes the contracts, failure models, and operational habits established earlier.

Every module contains a lesson, exercise, five-question quiz, complete reference solution, and explained quiz answers. Use only disposable data and local or dedicated practice services.
