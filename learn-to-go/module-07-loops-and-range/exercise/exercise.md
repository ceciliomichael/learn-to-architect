# Exercises: Repeat Work with For and Range

1. Print integers 1 through 10.
2. Count backward from 5 to 1.
3. Add integers 1 through 100 with a loop.
4. Range over `Go\u4e16\u754c` and print each byte index and rune.
5. Write an intentional infinite loop in disposable code, interrupt it, and repair its progress rule.
