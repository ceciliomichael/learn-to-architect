# Exercises: Strings, Bytes, Runes, and Unicode

1. Print an interpreted string containing a newline.
2. Print a raw two-line string.
3. Compare byte length and rune count for `Go` and `Go\u4e16\u754c`.
4. Print the numeric value and quoted form of rune `'G'`.
5. Explain why selecting a byte index is unsafe for an arbitrary Unicode character.
