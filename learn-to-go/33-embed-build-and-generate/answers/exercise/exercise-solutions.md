# Exercise Solutions: Embedded Files, Build Constraints, and Generation

`message/default.txt` contains `Welcome to Go.`. `message/message.go`:

```go
package message

import (
	_ "embed"
	"strings"
)

//go:generate go run ../internal/generate

//go:embed default.txt
var defaultMessage string

func Default() string { return strings.TrimSpace(defaultMessage) }
```

`message/message_test.go`:

```go
package message

import "testing"

func TestDefault(t *testing.T) {
	if got := Default(); got != "Welcome to Go." {
		t.Fatalf("Default() = %q", got)
	}
}
```

`platform/name_windows.go`:

```go
//go:build windows

package platform

func Name() string { return "windows" }
```

`platform/name_other.go`:

```go
//go:build !windows

package platform

func Name() string { return "non-windows" }
```

`internal/generate/main.go`:

```go
package main

import (
	"fmt"
	"os"
)

func main() {
	const content = "generated by the reviewed course tool\n"
	if err := os.WriteFile("generated.txt", []byte(content), 0o644); err != nil {
		fmt.Fprintln(os.Stderr, "generate file:", err)
		os.Exit(1)
	}
}
```

Run `go generate ./...` twice and verify that the second run produces no content difference.

Run:

```text
gofmt -w .
go generate ./...
go test ./...
$env:GOOS = "linux"
$env:GOARCH = "amd64"
go build -o app-linux-amd64 ./cmd/app
Remove-Item Env:GOOS,Env:GOARCH
```

Use `GOOS=linux GOARCH=amd64 go build -o app-linux-amd64 ./cmd/app` in a POSIX shell. Do not execute an artifact built for another target on the current machine.
