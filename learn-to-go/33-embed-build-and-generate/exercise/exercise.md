# Exercises: Embedded Files, Build Constraints, and Generation

1. Embed one UTF-8 default message and expose it through a function.
2. Add tests proving the resource is present in the built package.
3. Create Windows and non-Windows files returning different platform labels.
4. Add a harmless reviewed generator that creates a deterministic text file.
5. Cross-build a pure Go command for one additional target without running it.
