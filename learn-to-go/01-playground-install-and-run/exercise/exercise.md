# Exercises: Use the Go Playground and Install Go

1. Open the official Go Playground and run the lesson program.
2. Change its message to three separate printed lines.
3. Select Format and observe the indentation.
4. Remove one closing brace, read the compiler message, then repair it.
5. If Go is installed locally, save `main.go`, run it, format it, build it, and run the produced executable.
