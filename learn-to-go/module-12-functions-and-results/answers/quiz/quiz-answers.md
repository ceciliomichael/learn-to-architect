# Quiz Answers: Functions and Multiple Results

1. Yes. Each declared result is returned and assigned separately.
2. When they make several closely related results clearer without encouraging bare returns or hidden mutation.
3. A slice of the declared element type.
4. A function value that retains access to bindings from its surrounding scope.
5. Returning the calculation keeps it reusable and lets the caller choose output, storage, testing, or further work.
