# Quiz Answers: Packages, Main, Imports, and Output

1. `main` marks a package that can build as a command.
2. `func main()` is called when the command starts.
3. `fmt` identifies the package and `Println` identifies its exported function.
4. No. `Print` writes exactly the provided text; include `\n` when a line ending is needed.
5. Compilation fails. Go requires imports to have an actual use.
