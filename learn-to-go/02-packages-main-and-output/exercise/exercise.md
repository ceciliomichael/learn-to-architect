# Exercises: Packages, Main, Imports, and Output

1. Write a command that prints a learner name and course title on separate lines.
2. Use both `Print` and `Println` while keeping the final output readable.
3. Add a comment that explains a real output requirement.
4. Change `Println` to incorrect letter case, read the error, and repair it.
5. Add an unused import, observe the error, and remove the import.
