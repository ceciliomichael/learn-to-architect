# Quiz Answers: Links, Paths, Navigation, and Download Safety

1. **What is a relative URL resolved against?**

   The current document URL, or an explicit base URL when one is present.

2. **Why avoid click here as link text?**

   It does not describe the destination when read alone or scanned in a list of links.

3. **What does aria-current communicate?**

   It identifies the item representing the current page or current step.

4. **Must every external link open a new tab?**

   No. Normal same-context navigation is usually more predictable.

5. **Does download make a file trustworthy?**

   No. It only suggests download behavior and does not validate the file.
