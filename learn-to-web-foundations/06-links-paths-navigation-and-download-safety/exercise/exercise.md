# Exercise: Links, Paths, Navigation, and Download Safety

## Goal

Apply this module in a small page whose structure and behavior you can inspect in the browser.

## Steps

1. Create index.html and contact.html as siblings.
2. Add meaningful links between them.
3. Create an events folder and link to its index.html.
4. Add one same-page fragment link to a unique id.
5. Test every path from the actual page location, including browser Back behavior.

## Success check

The result must satisfy every step, remain understandable without visual styling alone, and produce no unexplained browser-console or missing-resource errors.
