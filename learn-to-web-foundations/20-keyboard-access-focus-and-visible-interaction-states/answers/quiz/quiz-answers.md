# Quiz Answers: Keyboard Access, Focus, and Visible Interaction States

1. **Which element should perform an in-page action?**

   A button.

2. **Which element should navigate to another URL?**

   A link with href.

3. **Why avoid positive tabindex?**

   It creates a separate focus order that is difficult to maintain and often contradicts reading order.

4. **What does tabindex=-1 allow?**

   Programmatic or fragment focus without adding the element to normal Tab order.

5. **When is a hidden skip link visible?**

   When it receives keyboard focus.
