# Exercise: Keyboard Access, Focus, and Visible Interaction States

## Goal

Apply this module in a page whose structure, styling, behavior, and accessibility can be inspected.

## Steps

1. Add a skip link targeting main content.
2. Use links for destinations and buttons for actions.
3. Keep DOM order equal to the intended reading order.
4. Create a visible focus indicator.
5. Complete every interaction using only a keyboard.

## Success check

Every step has observable browser evidence. The result remains usable with keyboard input, text zoom, narrow space, and unavailable optional resources whenever those conditions apply.
