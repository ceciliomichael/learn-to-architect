# Exercise: Headings, Landmarks, and Semantic Page Structure

## Goal

Apply this module in a small page whose structure and behavior you can inspect in the browser.

## Steps

1. Create a page with header, named primary navigation, main, and footer.
2. Give the page one h1.
3. Add two sections with h2 headings.
4. Use article only if one item could make sense independently.
5. Inspect the heading and landmark structure without relying on visual size.

## Success check

The result must satisfy every step, remain understandable without visual styling alone, and produce no unexplained browser-console or missing-resource errors.
