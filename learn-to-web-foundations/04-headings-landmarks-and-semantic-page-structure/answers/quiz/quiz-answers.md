# Quiz Answers: Headings, Landmarks, and Semantic Page Structure

1. **What belongs in main?**

   The document's primary content, excluding repeated site-wide navigation and footer content.

2. **How should heading levels be chosen?**

   By the nesting and hierarchy of the content.

3. **When is article appropriate?**

   When the content is independently meaningful or reusable, such as a post, product, or news item.

4. **Does every div need to become a section?**

   No. Use section only for a meaningful thematic group, normally with a heading.

5. **Why name a navigation landmark?**

   A name distinguishes its purpose, especially when a page has more than one navigation region.
