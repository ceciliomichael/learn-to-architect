# Quiz Answers: Create and Open a Local Web Page

1. **Why use the .html extension?**

   It identifies the file as an HTML document to tools and servers.

2. **What does the title element control?**

   It supplies the document title used by the browser tab, history, and bookmarks.

3. **Why include the viewport metadata?**

   It tells mobile browsers to use the device width instead of laying out a wider virtual page.

4. **Where can you inspect the parsed document?**

   In the browser developer tools Elements or Inspector panel.

5. **Why use a local server later?**

   It exercises HTTP paths and browser security behavior that direct file loading may not reproduce.
