# Exercise Solution: Create and Open a Local Web Page

```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>My first page</title>
  </head>
  <body>
    <h1>My first local web page</h1>
    <p>I created this document from plain text.</p>
  </body>
</html>
```

## Why this works

The browser tab says My first page and the page shows one heading and one paragraph.
