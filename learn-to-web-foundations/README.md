# Learn Web Foundations Step by Step

This course is the first stage of the Web Development Learning Path. It assumes no programming experience. You will begin with a local HTML document and finish with an accessible, responsive static site that you can inspect and publish.

## What belongs here

Web Foundations teaches the platform underneath every frontend framework:

- How clients, servers, URLs, DNS, HTTPS, requests, and responses fit together
- Semantic HTML for documents, navigation, media, tables, and forms
- CSS cascade, sizing, typography, layout, responsiveness, and motion
- Keyboard access, accessible names, native semantics, and careful ARIA
- Developer tools, progressive enhancement, performance, and clean publishing

JavaScript, TypeScript, React, Next.js, and Vue are deliberately absent. They become easier to understand after the document and style systems they control are familiar.

## How to study

For every module, type the example, predict the browser result, inspect the parsed elements and computed styles, complete the exercise, and answer the quiz before opening either answer file. Test with a keyboard, text zoom, narrow space, and unavailable optional resources whenever the lesson makes those conditions relevant.
