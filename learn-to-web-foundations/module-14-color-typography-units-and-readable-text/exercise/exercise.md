# Exercise: Color, Typography, Units, and Readable Text

## Goal

Apply this module in a page whose structure, styling, behavior, and accessibility can be inspected.

## Steps

1. Choose a system font stack.
2. Set a unitless body line height.
3. Limit article line length with ch.
4. Express status with text and a non-color cue.
5. Check contrast and 200 percent zoom in both supported color schemes.

## Success check

Every step has observable browser evidence. The result remains usable with keyboard input, text zoom, narrow space, and unavailable optional resources whenever those conditions apply.
