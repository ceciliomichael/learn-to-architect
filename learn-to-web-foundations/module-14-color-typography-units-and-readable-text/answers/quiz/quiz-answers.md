# Quiz Answers: Color, Typography, Units, and Readable Text

1. **Why is a unitless line-height useful?**

   Descendants inherit a multiplier appropriate to their own font size.

2. **What does rem reference?**

   The root element's font size.

3. **What does ch approximate?**

   The width of the zero glyph, useful as a rough text-line measure.

4. **Can color alone identify an error?**

   No. Provide another perceivable cue and explanatory text.

5. **Why test text zoom?**

   Users may enlarge text, and content must remain readable without clipping or loss.
