# Quiz Answers: Custom Properties, Reusable Components, and Design Consistency

1. **What is a CSS custom property?**

   A named property value that participates in the cascade and is read with var().

2. **Why prefer purpose names such as color-accent?**

   The meaning can stay stable when the literal visual value changes.

3. **What should a variant contain?**

   Only the explicit differences from the usable base component.

4. **Do custom properties ignore the cascade?**

   No. They inherit and cascade like other properties.

5. **When should a value become a shared token?**

   When it represents a repeated, meaningful design decision.
