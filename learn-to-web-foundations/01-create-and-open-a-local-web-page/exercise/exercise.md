# Exercise: Create and Open a Local Web Page

## Goal

Apply this module in a small page whose structure and behavior you can inspect in the browser.

## Steps

1. Create a folder named first-site and place index.html inside it.
2. Type the complete document without copying it blindly.
3. Change the title and visible heading to your own wording.
4. Open developer tools and locate the h1 element.
5. Deliberately remove one closing angle bracket, observe the parsed result, then repair it.

## Success check

The result must satisfy every step, remain understandable without visual styling alone, and produce no unexplained browser-console or missing-resource errors.
