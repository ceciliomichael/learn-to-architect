# Exercise: Accessible Names, Forms, Tables, and Careful ARIA

## Goal

Apply this module in a page whose structure, styling, behavior, and accessibility can be inspected.

## Steps

1. Create a labeled email field with persistent help.
2. Show a visible invalid example with correction text.
3. Associate help and error text with the field.
4. Use a native submit button.
5. Inspect name, role, description, state, and value.

## Success check

Every step has observable browser evidence. The result remains usable with keyboard input, text zoom, narrow space, and unavailable optional resources whenever those conditions apply.
