# Quiz Answers: Accessible Names, Forms, Tables, and Careful ARIA

1. **What should be preferred before ARIA?**

   The native HTML element with the required semantics and behavior.

2. **Does role=button create keyboard behavior?**

   No.

3. **What can aria-describedby associate?**

   Supplementary help or error descriptions.

4. **What does aria-invalid communicate?**

   That the current value failed validation.

5. **Can ARIA repair poor focus order or missing keyboard logic?**

   No. Those behaviors must be implemented correctly.
