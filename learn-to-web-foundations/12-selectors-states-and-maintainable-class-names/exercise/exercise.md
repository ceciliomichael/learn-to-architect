# Exercise: Selectors, States, and Maintainable Class Names

## Goal

Apply this module in a small page whose structure and behavior you can inspect in the browser.

## Steps

1. Create one reusable navigation-link class.
2. Add distinct hover and focus-visible treatments.
3. Style the current link through aria-current.
4. Keep selectors no deeper than needed.
5. Test pointer, keyboard, visited, and narrow-screen behavior.

## Success check

The result must satisfy every step, remain understandable without visual styling alone, and produce no unexplained browser-console or missing-resource errors.
