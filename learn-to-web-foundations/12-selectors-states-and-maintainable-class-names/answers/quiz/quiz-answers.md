# Quiz Answers: Selectors, States, and Maintainable Class Names

1. **Why are classes usually preferred for reusable styling?**

   They can be applied to several elements and generally keep specificity manageable.

2. **What does :focus-visible target?**

   Elements whose focus should receive a visible indicator according to input and browser heuristics.

3. **Can hover be the only way to expose content?**

   No. Touch and keyboard users may never trigger it.

4. **Why avoid long descendant selectors?**

   They couple styles to exact markup nesting and create difficult specificity.

5. **Should a custom focus style reduce visibility?**

   No. It must remain clearly perceivable against surrounding colors.
