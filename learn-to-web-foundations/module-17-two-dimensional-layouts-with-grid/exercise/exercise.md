# Exercise: Two-Dimensional Layouts with Grid

## Goal

Apply this module in a page whose structure, styling, behavior, and accessibility can be inspected.

## Steps

1. Create a list of at least four categories.
2. Use Grid with a gap.
3. Use repeat and minmax for responsive tracks.
4. Prevent a track from becoming wider than its container.
5. Verify source, reading, and visual order agree.

## Success check

Every step has observable browser evidence. The result remains usable with keyboard input, text zoom, narrow space, and unavailable optional resources whenever those conditions apply.
