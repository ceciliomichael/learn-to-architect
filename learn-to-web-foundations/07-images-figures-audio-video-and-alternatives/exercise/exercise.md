# Exercise: Images, Figures, Audio, Video, and Alternatives

## Goal

Apply this module in a small page whose structure and behavior you can inspect in the browser.

## Steps

1. Add one informative image with accurate dimensions and useful alt text.
2. Add one decorative image with alt="".
3. Group an image and caption with figure and figcaption.
4. Add a controlled video with a caption track.
5. Provide a transcript link and test the page with images disabled.

## Success check

The result must satisfy every step, remain understandable without visual styling alone, and produce no unexplained browser-console or missing-resource errors.
