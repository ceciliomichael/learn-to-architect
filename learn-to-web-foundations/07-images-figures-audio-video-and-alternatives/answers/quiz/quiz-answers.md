# Quiz Answers: Images, Figures, Audio, Video, and Alternatives

1. **What alt value should a decorative image use?**

   An empty value, alt="", so it is ignored as content.

2. **Why include image width and height?**

   They establish an aspect ratio and reduce layout movement while the image loads.

3. **What does figcaption describe?**

   The figure as a whole, not necessarily a word-for-word replacement for image alt text.

4. **Why avoid autoplaying sound?**

   Unexpected audio interferes with user control and can create accessibility problems.

5. **What does a captions track include?**

   Synchronized speech and important non-speech audio information.
