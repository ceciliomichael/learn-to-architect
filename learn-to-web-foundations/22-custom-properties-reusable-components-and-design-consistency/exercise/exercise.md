# Exercise: Custom Properties, Reusable Components, and Design Consistency

## Goal

Apply this module in a page whose structure, styling, behavior, and accessibility can be inspected.

## Steps

1. Define purposeful color and spacing custom properties.
2. Build one native button component class.
3. Add one variant without duplicating the entire base.
4. Test all interactive states and contrast.
5. Change one property and confirm the intended components update.

## Success check

Every step has observable browser evidence. The result remains usable with keyboard input, text zoom, narrow space, and unavailable optional resources whenever those conditions apply.
