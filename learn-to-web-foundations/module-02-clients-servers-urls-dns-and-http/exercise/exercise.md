# Exercise: Clients, Servers, URLs, DNS, and HTTP

## Goal

Apply this module in a small page whose structure and behavior you can inspect in the browser.

## Steps

1. Break the example URL into its six labeled parts.
2. Explain which part DNS uses and which parts the HTTP request carries.
3. Find the request method and response status.
4. Explain the Content-Type header.
5. Use the Network panel on a public non-private page and identify one document request without copying private values.

## Success check

The result must satisfy every step, remain understandable without visual styling alone, and produce no unexplained browser-console or missing-resource errors.
