# Exercise: Paragraphs, Lists, Quotations, and Code Text

## Goal

Apply this module in a small page whose structure and behavior you can inspect in the browser.

## Steps

1. Write one introductory paragraph.
2. Create an unordered materials list.
3. Create an ordered three-step procedure.
4. Add a keyboard shortcut with kbd.
5. Display one HTML tag as code without letting the browser parse it.

## Success check

The result must satisfy every step, remain understandable without visual styling alone, and produce no unexplained browser-console or missing-resource errors.
