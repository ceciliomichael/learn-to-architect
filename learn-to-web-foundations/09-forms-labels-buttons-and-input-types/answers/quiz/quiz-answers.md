# Quiz Answers: Forms, Labels, Buttons, and Input Types

1. **Why is placeholder not a label?**

   It disappears during input, may have poor contrast, and does not reliably provide a persistent control name.

2. **What does a control name do?**

   It identifies the key used for that control's submitted value.

3. **Why use type=email?**

   It communicates the expected value and can provide email-oriented input and native validation.

4. **What is the default type of button inside a form?**

   submit.

5. **What does autocomplete describe?**

   The semantic purpose of a field so the browser can offer appropriate stored information.
