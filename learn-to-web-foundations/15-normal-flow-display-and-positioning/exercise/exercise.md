# Exercise: Normal Flow, Display, and Positioning

## Goal

Apply this module in a page whose structure, styling, behavior, and accessibility can be inspected.

## Steps

1. Build three sections that stack in normal flow.
2. Use inline display for an in-sentence label.
3. Create one positioned decorative mark inside a relative container.
4. Confirm the decoration is hidden from assistive technology.
5. Test zoom and long content for overlap.

## Success check

Every step has observable browser evidence. The result remains usable with keyboard input, text zoom, narrow space, and unavailable optional resources whenever those conditions apply.
