# Exercise: Responsive Design, Media Queries, and Container Queries

## Goal

Apply this module in a page whose structure, styling, behavior, and accessibility can be inspected.

## Steps

1. Create a narrow single-column base.
2. Add a layout change based on available component width.
3. Use a viewport media query only for a page-wide need.
4. Include one user-preference query.
5. Test zoom, portrait, landscape, and a narrow containing column.

## Success check

Every step has observable browser evidence. The result remains usable with keyboard input, text zoom, narrow space, and unavailable optional resources whenever those conditions apply.
