# Quiz Answers: Transforms, Transitions, Animation, and Reduced Motion

1. **What should motion communicate?**

   A meaningful change, relationship, feedback, or carefully limited attention cue.

2. **Why list transitioned properties explicitly?**

   transition: all can animate unintended changes and make performance or behavior harder to predict.

3. **What does prefers-reduced-motion represent?**

   A user preference to minimize nonessential motion.

4. **Should reduced motion remove information?**

   No. Preserve the state and meaning through another presentation.

5. **Is styling enough to implement a disclosure button?**

   No. Script must synchronize behavior and accessible state.
