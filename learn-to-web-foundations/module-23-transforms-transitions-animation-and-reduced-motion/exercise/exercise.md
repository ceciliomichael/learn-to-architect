# Exercise: Transforms, Transitions, Animation, and Reduced Motion

## Goal

Apply this module in a page whose structure, styling, behavior, and accessibility can be inspected.

## Steps

1. Create a two-state visual transition.
2. Keep duration short and scope properties explicitly.
3. Add a reduced-motion override.
4. Ensure the final state remains understandable without movement.
5. Record the future script contract for synchronizing hidden and aria-expanded.

## Success check

Every step has observable browser evidence. The result remains usable with keyboard input, text zoom, narrow space, and unavailable optional resources whenever those conditions apply.
