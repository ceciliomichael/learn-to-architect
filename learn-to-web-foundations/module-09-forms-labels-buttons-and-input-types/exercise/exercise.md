# Exercise: Forms, Labels, Buttons, and Input Types

## Goal

Apply this module in a small page whose structure and behavior you can inspect in the browser.

## Steps

1. Create a workshop registration form.
2. Add labeled name and email controls with autocomplete.
3. Add a checkbox whose label explains the choice.
4. Give every submitted control a name.
5. Add an explicit submit button and inspect keyboard order.

## Success check

The result must satisfy every step, remain understandable without visual styling alone, and produce no unexplained browser-console or missing-resource errors.
