# Exercise Solution: Tables for Tabular Data

```html
<table>
  <caption>Garden opening hours</caption>
  <thead>
    <tr>
      <th scope="col">Day</th>
      <th scope="col">Opens</th>
      <th scope="col">Closes</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">Monday</th>
      <td>09:00</td>
      <td>18:00</td>
    </tr>
    <tr>
      <th scope="row">Saturday</th>
      <td>10:00</td>
      <td>16:00</td>
    </tr>
  </tbody>
</table>
```

## Why this works

Every time cell is associated with both a day row header and an opens or closes column header.
