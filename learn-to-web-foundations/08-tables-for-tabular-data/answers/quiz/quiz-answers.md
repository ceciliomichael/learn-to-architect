# Quiz Answers: Tables for Tabular Data

1. **When is table appropriate?**

   When data cells have meaningful row and column relationships.

2. **What does scope=col mean?**

   The header applies to cells in its column.

3. **What does caption provide?**

   A title or concise identification for the table.

4. **Do thead and tbody create header relationships by themselves?**

   No. th and its associations define the headers.

5. **Why not use tables for page layout?**

   They misrepresent structure and produce rigid, confusing reading and navigation behavior.
