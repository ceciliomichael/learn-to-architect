# Exercise: The Box Model, Sizing, Overflow, and Spacing

## Goal

Apply this module in a page whose structure, styling, behavior, and accessibility can be inspected.

## Steps

1. Apply border-box sizing to all elements and pseudo-elements.
2. Create a card with padding and a visible border.
3. Allow it to fill narrow space but cap its maximum width.
4. Center it with logical margins.
5. Test 200 percent text zoom and one very long unbroken string.

## Success check

Every step has observable browser evidence. The result remains usable with keyboard input, text zoom, narrow space, and unavailable optional resources whenever those conditions apply.
