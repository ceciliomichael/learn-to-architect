# Quiz Answers: The Box Model, Sizing, Overflow, and Spacing

1. **What surrounds the content box?**

   Padding, then border, then margin.

2. **What does border-box change?**

   Declared width and height include padding and border.

3. **Why combine width: 100% and max-width?**

   The box can shrink on small containers without growing beyond a readable limit.

4. **Does overflow: hidden solve overflowing content?**

   It may hide the symptom by making content unavailable.

5. **Why avoid fixed text-container heights?**

   Text size, wrapping, language, and user settings can require more space.
