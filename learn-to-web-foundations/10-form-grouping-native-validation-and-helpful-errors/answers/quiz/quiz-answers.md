# Quiz Answers: Form Grouping, Native Validation, and Helpful Errors

1. **What labels a fieldset?**

   Its legend element.

2. **Why do related radio buttons share a name?**

   The shared name makes them one mutually exclusive submitted choice.

3. **Can required be trusted by the server?**

   No. A client can remove or bypass it.

4. **What makes an error actionable?**

   It identifies the affected field, explains the violated rule, and tells the user how to correct it.

5. **Why not communicate invalid state only with red color?**

   Some users cannot perceive that distinction, and color alone gives no textual meaning.
