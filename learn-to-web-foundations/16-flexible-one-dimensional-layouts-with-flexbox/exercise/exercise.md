# Exercise: Flexible One-Dimensional Layouts with Flexbox

## Goal

Apply this module in a page whose structure, styling, behavior, and accessibility can be inspected.

## Steps

1. Create a semantic list of three cards.
2. Make the list a wrapping flex container.
3. Give each item a flexible 16rem basis.
4. Use gap instead of margins between items.
5. Test long words, narrow widths, and keyboard reading order.

## Success check

Every step has observable browser evidence. The result remains usable with keyboard input, text zoom, narrow space, and unavailable optional resources whenever those conditions apply.
