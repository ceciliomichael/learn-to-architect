# Quiz Answers: Flexible One-Dimensional Layouts with Flexbox

1. **What kind of layout is Flexbox designed around?**

   One main dimension at a time, either row or column.

2. **What does flex-wrap permit?**

   Items can continue on additional flex lines.

3. **What does gap create?**

   Consistent space between flex or grid items without outer-edge margins.

4. **Why might a flex item need min-width: 0?**

   Its default minimum content size can otherwise prevent shrinking.

5. **Should visual order contradict source order?**

   No. Reading and keyboard navigation generally follow source order.
