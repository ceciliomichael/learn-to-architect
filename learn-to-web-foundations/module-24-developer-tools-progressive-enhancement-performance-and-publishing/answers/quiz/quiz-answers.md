# Quiz Answers: Developer Tools, Progressive Enhancement, Performance, and Publishing

1. **Which panel shows HTTP requests and failed resources?**

   The Network panel.

2. **What is progressive enhancement?**

   Deliver a usable baseline, then add enhancements when supported without breaking the core task.

3. **Why test the deployed URL separately?**

   Base paths, headers, case sensitivity, caching, and server behavior can differ from local work.

4. **Should a static site contain secrets?**

   No. Browser-delivered files are available to visitors.

5. **What should a release checklist include?**

   Content, links, accessibility, responsive layouts, performance, security, file contents, deployment behavior, and rollback or update instructions.
