# Exercise: Tables for Tabular Data

## Goal

Apply this module in a small page whose structure and behavior you can inspect in the browser.

## Steps

1. Create a captioned table for three workshop dates.
2. Add clear column headers.
3. If each workshop name labels its row, mark it as a row header.
4. Use tbody for the data rows.
5. Read each cell as header plus value and simplify any relationship that is hard to explain.

## Success check

The result must satisfy every step, remain understandable without visual styling alone, and produce no unexplained browser-console or missing-resource errors.
