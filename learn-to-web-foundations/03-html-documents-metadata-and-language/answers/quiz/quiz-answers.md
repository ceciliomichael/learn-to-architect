# Quiz Answers: HTML Documents, Metadata, and Language

1. **What does the doctype do?**

   It asks browsers to parse the page in standards mode.

2. **Where does visible page content normally belong?**

   Inside the body element.

3. **Why declare a document language?**

   User agents and assistive tools use it for pronunciation, translation, spelling, and other language-sensitive behavior.

4. **Are title and h1 interchangeable?**

   No. title identifies the document outside its content area, while h1 is a visible heading in the document.

5. **Why place charset metadata early?**

   The browser should know the encoding before interpreting later text bytes.
