# Exercise: HTML Documents, Metadata, and Language

## Goal

Apply this module in a small page whose structure and behavior you can inspect in the browser.

## Steps

1. Create a complete document for a community garden contact page.
2. Set the correct document language.
3. Write a unique title that includes the site name.
4. Add a one-sentence description that matches the visible content.
5. Confirm the visible h1 is not replaced by the title element.

## Success check

The result must satisfy every step, remain understandable without visual styling alone, and produce no unexplained browser-console or missing-resource errors.
