# Exercise: CSS Rules, the Cascade, Inheritance, and Specificity

## Goal

Apply this module in a small page whose structure and behavior you can inspect in the browser.

## Steps

1. Create an external styles.css and link it from HTML.
2. Set a body font and text color that descendants inherit.
3. Style a notice class with border and padding.
4. Add two same-specificity notice rules and predict which source order wins.
5. Confirm the result in computed styles, then remove the unnecessary conflict.

## Success check

The result must satisfy every step, remain understandable without visual styling alone, and produce no unexplained browser-console or missing-resource errors.
