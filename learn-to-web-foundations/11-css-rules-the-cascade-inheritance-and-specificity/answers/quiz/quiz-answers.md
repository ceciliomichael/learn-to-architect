# Quiz Answers: CSS Rules, the Cascade, Inheritance, and Specificity

1. **What are the two main parts of a CSS rule?**

   A selector and a declaration block.

2. **What happens to an invalid declaration?**

   The browser normally ignores that declaration while continuing to parse the rest.

3. **When two otherwise equal rules have equal specificity, what commonly decides?**

   The later declaration in source order.

4. **Do margins inherit by default?**

   No.

5. **Where can the winning declaration be inspected?**

   In the browser's Styles and Computed developer-tool panels.
