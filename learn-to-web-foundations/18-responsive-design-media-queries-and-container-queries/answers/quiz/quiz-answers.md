# Quiz Answers: Responsive Design, Media Queries, and Container Queries

1. **What does mobile-first mean here?**

   Start with a simple narrow layout and add enhancements for more available space.

2. **What should determine a width breakpoint?**

   The content and layout behavior, not a device brand.

3. **What does a container query inspect?**

   The size or features of an established query container.

4. **What does prefers-reduced-motion express?**

   A user preference to reduce nonessential motion.

5. **Must the page fail without container-query support?**

   No. The base layout should remain usable.
