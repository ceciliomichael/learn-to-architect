# Quiz Answers: Two-Dimensional Layouts with Grid

1. **What can Grid control together?**

   Rows and columns.

2. **What does fr represent?**

   A fraction of remaining space in the grid container.

3. **What does auto-fit do in the example?**

   It creates as many tracks as fit and collapses unused tracks.

4. **Why use min(100%, 15rem) in the minimum?**

   It prevents the minimum track size from overflowing a container narrower than 15rem.

5. **Does visual grid placement change document reading order?**

   No. Source order remains critical.
