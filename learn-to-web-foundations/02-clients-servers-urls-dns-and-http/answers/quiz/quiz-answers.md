# Quiz Answers: Clients, Servers, URLs, DNS, and HTTP

1. **What does DNS resolve?**

   It resolves a host name to information needed to reach the named service, commonly IP addresses.

2. **Does the URL fragment go to the server in an HTTP request?**

   No. The browser handles it locally.

3. **What does an HTTP method express?**

   It expresses the intended request operation, such as retrieving or submitting a representation.

4. **What does Content-Type describe?**

   It describes the media type and optional character encoding of the message body.

5. **What extra protection does HTTPS add?**

   It authenticates the server certificate relationship and encrypts and integrity-protects traffic in transit.
