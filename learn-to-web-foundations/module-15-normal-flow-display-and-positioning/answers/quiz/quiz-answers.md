# Quiz Answers: Normal Flow, Display, and Positioning

1. **What does normal flow provide?**

   Default block stacking and inline line flow that adapts to content.

2. **What does display: none do?**

   It removes the element from layout and normally from the accessibility tree.

3. **Does position: relative remove an element's original space?**

   No.

4. **What establishes an absolute element's reference box?**

   Its containing block, often the nearest positioned ancestor.

5. **Why limit absolute positioning for content?**

   Removed content does not reserve space and can overlap variable text.
