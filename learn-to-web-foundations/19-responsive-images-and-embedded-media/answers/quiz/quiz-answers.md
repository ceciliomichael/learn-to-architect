# Quiz Answers: Responsive Images and Embedded Media

1. **What does srcset with w descriptors provide?**

   Alternative intrinsic image widths from which the browser can choose.

2. **What does sizes estimate?**

   The rendered slot width under listed media conditions.

3. **When is picture appropriate?**

   For art direction, format choice, or source conditions where different image content is needed.

4. **Why set iframe title?**

   It identifies the embedded content to assistive technology.

5. **What costs can third-party embeds introduce?**

   Network, performance, privacy, security, and availability costs.
