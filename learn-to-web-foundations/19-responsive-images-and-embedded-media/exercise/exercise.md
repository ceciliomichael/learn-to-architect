# Exercise: Responsive Images and Embedded Media

## Goal

Apply this module in a page whose structure, styling, behavior, and accessibility can be inspected.

## Steps

1. Create three sizes of the same practice image.
2. Write srcset width candidates and an honest sizes value.
3. Keep width, height, and alt attributes.
4. Apply responsive media CSS.
5. Inspect which source loads at several viewport widths and pixel densities.

## Success check

Every step has observable browser evidence. The result remains usable with keyboard input, text zoom, narrow space, and unavailable optional resources whenever those conditions apply.
