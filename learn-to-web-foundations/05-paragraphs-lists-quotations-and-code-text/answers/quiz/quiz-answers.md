# Quiz Answers: Paragraphs, Lists, Quotations, and Code Text

1. **When should ol be used?**

   When sequence, priority, or rank changes the meaning.

2. **What element contains each list item?**

   The li element.

3. **What does blockquote represent?**

   A quotation presented as a block from another source.

4. **Why combine pre and code?**

   code gives technical meaning while pre preserves line breaks and spacing.

5. **How is a literal less-than sign written in displayed HTML code?**

   Use the &lt; character reference.
