## Question 2

What is displayed?

```text
Set count = 2

WHILE count <= 5
    Display count
    Set count = count + 1
END WHILE
```

## Question 3

What is a priming read? Why does this design need the first Input before the WHILE?

```text
Set total = 0
Input score
WHILE score <> -1
    Set total = total + score
    Input score
END WHILE
```

## Question 4
