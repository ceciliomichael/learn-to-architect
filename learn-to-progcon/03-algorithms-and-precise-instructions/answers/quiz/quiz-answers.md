# Module 03 Quiz Answers

## Answer 1

An algorithm is a clear set of steps that solves a problem or completes a task.

Strong answers may also mention that the steps should be ordered, precise, and finite.

## Answer 2

**B.** Ordered, precise, finite.

## Answer 3

Any two (or more) of these reasons are good:

- "Get the scores" does not say how many scores or name them.
- "Do the average thing" does not state the formula or where to store the result.
- "Maybe show it" is not a definite output step.
- The method is not precise enough for two people to get the same result every time.
- A clear stop is missing.

## Answer 5

An algorithm is the method: the ordered steps that solve the problem. A program is a form of those instructions written so a computer can perform them, usually in a programming language. You can write and check an algorithm before you choose language syntax.
