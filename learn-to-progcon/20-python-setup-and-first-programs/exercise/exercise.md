# Module 20 Exercises

Complete these tasks in a file such as `main.py`. Run the file after each small change.

## Warm-up: Change a working program

Start with:

```python
print("Good morning")
```

1. Change the message to greet yourself by name.
2. Add a second line that prints `I ran my first Python program.`
3. Add a comment above the prints that states the program purpose.
4. Run the file and confirm both lines appear in order.

## Guided exercise: Translate a receipt header

Pseudocode:

```text
START
    OUTPUT "Corner Shop"
    OUTPUT "Receipt"
    OUTPUT "Thank you"
END
```

1. Write the matching Python program with three `print` calls.
2. Run it.
3. Add one comment.
4. Confirm the output order matches the design.

## Independent exercise: Design then code

1. Write a short sequential pseudocode design that outputs four lines:
   - your name
   - a place you want to visit
   - one programming concept you have learned
   - `End of introduction.`
2. Translate the design into Python.
3. Run it and check that exactly four lines print.
4. Keep quotation marks out of the visible output text.

## Debugging task

Fix this program so it prints both messages:

```python
Print("Starting now")
print("Practice makes progress)
print("Finished")
```

- [ ] I can separate source from output.

- [ ] I translated a tiny design into Python.
