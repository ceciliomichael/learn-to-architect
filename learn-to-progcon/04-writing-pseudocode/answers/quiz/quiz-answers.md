# Module 04 Quiz Answers

## Answer 1

Pseudocode is used to write a clear, structured design of an algorithm in plain language and common keywords. It helps people clarify logic, communicate the method, desk-check it, and prepare for later implementation. It is not meant as a full programming language.

## Answer 2

1. `INPUT` -> **C** Read a value into a name  
2. `OUTPUT` -> **A** Show a value or message  
3. `SET` -> **D** Store a value in a name  
4. `START` / `END` -> **B** Mark the beginning and end of the design  

## Answer 3
