## Answer 2

**B.** Hardware is physical devices. Software is instructions and related data.

Hardware is what you can touch. Software tells hardware what to do.

## Answer 3

A program is a set of instructions that a computer can follow to complete a task.

Strong answers may also note that people write those instructions and that tools help prepare them for the machine. The core idea is still: ordered instructions for a task.

## Answer 4

A. The three numbered steps are **source instructions** (the design text a person wrote).  
B. The value `27` on the screen is a **result** (output produced when the instructions run with input `25`).

## Answer 5

The **cashier** is mainly in a **user environment**: they use the finished app to complete a sale and see screens and totals, not the project's source files.

The **developer** is mainly in a **programming environment**: they open project text files, change the tax instructions, and test the fix.
