# Quiz Answers: Browser Runtime, Project Setup, and Generated JavaScript

1. It selects Vite's plain browser TypeScript template and does not add React, Vue, or another UI framework.
2. HTTP better represents deployed module loading and URL behavior. Vite can also transform TypeScript, resolve imports, and report development errors.
3. Node.js runs development tools. Vite serves and builds the project. TypeScript checks types. The browser executes the resulting JavaScript and supplies DOM APIs.
4. The generic type describes the expected result when an element is found. It cannot guarantee that runtime HTML contains the element.
5. `src` is authored source, `node_modules` is installed dependency output, and `dist` is generated deployment output.
