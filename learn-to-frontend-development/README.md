# Learn Frontend Development with TypeScript

This course is the framework-free browser stage of the Web Development Learning Path. Complete Web Foundations and Learn TypeScript first.

You will use TypeScript with the actual browser platform: DOM nodes, events, forms, navigation, HTTP, runtime validation, storage, security policies, files, scheduling, testing, accessibility, performance, service workers, and deployment. React and Vue are intentionally absent so their abstractions will have concrete meaning later.

## Required prerequisites

- [Web Foundations](../learn-to-web-foundations/README.md)
- [Learn TypeScript](../learn-to-typescript/README.md)

## Local course project

Module 01 creates one Vite `vanilla-ts` project named `browser-course`. Keep using that project throughout the course. Install a package only when a module introduces its responsibility, and keep every installation recorded in `package.json` and `package-lock.json`.

The course uses Node.js 24 LTS for local tooling. Node.js runs Vite, TypeScript, and test tools. The application code still runs in the browser.

After Module 26, choose [React](../learn-to-react/README.md) or [Vue](../learn-to-vue/README.md). React learners may continue to Next.js afterward. See the complete [Web Development Learning Path](../learning-paths/web-development.md).
