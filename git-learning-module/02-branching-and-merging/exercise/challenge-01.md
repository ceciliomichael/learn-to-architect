# Challenge 01: Creating a Parallel Universe

**Scenario:**
You are working on the main timeline of your application, but you have been tasked with building a brand new authentication feature. You do not want to risk breaking the stable code on the main timeline.

Your objective is to create a new, isolated environment named `feature-auth` and step into it.

**Tasks:**
1. Determine your current branch.
2. Create a new branch named `feature-auth`.
3. Switch your working directory to the `feature-auth` branch.
4. Verify that you have successfully moved into the new branch.
