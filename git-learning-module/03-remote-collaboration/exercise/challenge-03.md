# Challenge 03: Safe Reconnaissance

## Scenario
You just arrived at work. Your teammate has pushed a massive amount of new code to the `main` branch on the remote repository.

You have uncommitted changes in your local Working Directory. You want to download your teammate's new snapshots to see what they did, but you are terrified of forcefully merging their code into your uncommitted work and triggering massive merge conflicts.

## Task
Explain the safest way to download these changes from the `origin` remote without aggressively merging them into your current Working Directory. Provide the exact command you would use.
