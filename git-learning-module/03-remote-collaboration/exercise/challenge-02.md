# Challenge 02: Uploading the Album

## Scenario
You have successfully linked your local repository to the remote library known as `origin`. Now, you want to upload your latest commits from your local `main` branch to the remote repository.

Because this is the very first time you are pushing this branch, you want Git to remember this connection for future, simpler pushes.

## Task
Write the Git command to push the `main` branch to the `origin` remote, ensuring that you set the upstream tracking connection at the same time.
