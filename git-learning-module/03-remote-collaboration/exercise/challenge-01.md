# Challenge 01: Linking the Library

## Scenario
You have been working on your local Photography Studio (your local repository) and are now ready to back it up to the Global Library (a remote repository).

Your task is to tell your local Git environment how to connect to this remote locker.

## Task
Provide the exact Git command required to register a new remote connection.

Use the standard industry nickname for the primary remote and assume the remote URL is `https://github.com/company/project.git`.
