# The Git & Version Control Academy

Welcome to the Git Academy! Whether you write TypeScript, Python, Rust, or Go, there is one universal tool that absolutely every software engineer must master: **Git**.

## What is Git?

At its core, Git is a localized time machine for your codebase. It is not just a backup system; it is a parallel universe simulator. It allows you to:
1. **Save Snapshots:** Take permanent, unbreakable photographs of your code at a specific moment in time.
2. **Rewind Mistakes:** Instantly undo catastrophic errors by rolling the entire project back to a safe snapshot.
3. **Branch Realities:** Create parallel universes (branches) to experiment with new features without ever breaking the main, stable application.
4. **Collaborate Securely:** Merge work from dozens of developers around the world seamlessly.

## How to Complete a Module

Inside every module folder, you will find:
1. **`README.md`:** The core lesson. Read this entirely before touching any code.
2. **`exercise/`:** Hands-on lab challenges to practice the commands in a real repository.
3. **`quiz/`:** Conceptual questions to test your architectural understanding of Git's mechanics.
4. **`answers/`:** Exhaustive, production-grade solutions explaining the exact commands and why they work.
