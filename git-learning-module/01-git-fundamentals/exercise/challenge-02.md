# Challenge 02: Framing the Shot

## Scenario
You have added several new files to your project. However, you only want to group specific files together for your next snapshot. In this challenge, you will selectively move files into the Staging Area.

## Instructions
1. Ensure you are in your initialized Git repository.
2. Create three new files named `index.html`, `app.js`, and `notes.txt`.
3. Use the appropriate Git command to add ONLY `index.html` and `app.js` to the Staging Area.
4. Verify the status of your repository to ensure the correct files are staged and the incorrect file is omitted.
