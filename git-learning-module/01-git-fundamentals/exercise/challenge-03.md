# Challenge 03: The First Snapshot

## Scenario
You have carefully framed your files in the Staging Area. Now, you need to save them permanently into your repository history and review the timeline.

## Instructions
1. Ensure `index.html` and `app.js` are still in your Staging Area from the previous challenge.
2. Create a permanent snapshot of these files with a descriptive message.
3. Review the history of your repository to verify the snapshot was recorded successfully.
