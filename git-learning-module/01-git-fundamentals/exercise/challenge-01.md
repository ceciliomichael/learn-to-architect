# Challenge 01: Initializing and Checking Status

## Scenario
You are starting a brand new project and need to set up your version control system. In this challenge, you will initialize a Git repository and observe how Git tracks newly created files in your working directory.

## Instructions
1. Create a new directory for your project and navigate into it.
2. Initialize the folder as a Git repository whose initial branch is named `main`.
3. Create a new file named `readme.txt`.
4. Check the status of your repository.
