# Plan for Vue

## Boundary

This 24-module course follows Frontend Development with TypeScript and teaches Vue 3 with TypeScript. It uses Single-File Components for application work and explains Composition API behavior without requiring React knowledge.

- [x] 01: Vue Playground, create-vue, and the Application Root

- [x] 03: Template Syntax and Safe Expressions

- [x] 05: Computed Values and Derived State
- [x] 06: Class and Style Bindings
- [x] 07: Conditional and List Rendering with Stable Keys
- [x] 08: Events, Modifiers, and Handler Types
- [x] 09: Forms and v-model
- [x] 10: Lifecycle Hooks and Template Refs
- [x] 11: Component Registration and Composition

- [x] 16: Composables and Reusable Stateful Logic
- [x] 17: Watchers, watchEffect, and External Synchronization
- [x] 18: Provide, Inject, and Dependency Boundaries
- [x] 19: Vue Router and Navigation
- [x] 20: Pinia and Shared Application State

- [x] 23: Accessibility, Security, and Measured Performance
