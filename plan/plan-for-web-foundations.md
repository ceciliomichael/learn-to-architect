# Plan for Web Foundations

## Boundary

This 24-module course begins with no programming prerequisite and ends with a complete, accessible, responsive static site. It does not introduce a JavaScript framework. Programming begins in the TypeScript course after learners understand the documents and styles scripts will later control.

- [x] 20: Keyboard Access, Focus, and Visible Interaction States

## Dependency rules

1. Semantic HTML comes before styling.
2. The cascade and box model come before Flexbox and Grid.
3. Responsive layout comes before responsive media.
4. Native semantics come before ARIA.
5. Animation includes reduced-motion behavior from its first lesson.
