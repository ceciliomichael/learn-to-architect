# Exercises: Build Expressions and Decisions

## Exercise 1: Calculate results

1. Show each title, stored price, and a price after a 10 percent increase.
2. Name the calculated column `increased_price`.
3. Order from highest calculated price to lowest.
4. Prove the stored price did not change by selecting it again.

## Exercise 2: Build descriptions

Return one column that reads like `First Steps in SQL by Mira Chen`. Name it `book_description`.

## Exercise 3: Classify rows

Use `CASE` to label books:

- `Low` when price is below 23
- `Middle` when price is below 32 after the first test
- `High` otherwise

Show title, price, and label. Order by price and explain each boundary.
