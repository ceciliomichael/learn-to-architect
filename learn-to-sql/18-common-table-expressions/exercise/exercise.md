# Exercises: Organize Queries with Common Table Expressions

## Exercise 1: Name a filtered set

Create `affordable_books` for books below 25, then select its title and price in price order.

## Exercise 2: Build two steps

In `shop.db`, create one CTE with order totals and a second that keeps totals at least a threshold you choose. Return customer, order, and total.

## Exercise 3: State each grain

Write one sentence describing the grain of each CTE and the final result from Exercise 2. Confirm the selected columns obey those grains.
