# Exercises: Combine Result Sets

## Exercise 1: Compare UNION forms

Combine book authors with order customer names using `UNION`, then `UNION ALL`. Count and explain the difference.

## Exercise 2: Find overlap and difference

Create two temporary practice tables with one text column and overlapping names. Use `INTERSECT` and both directions of `EXCEPT`.

## Exercise 3: Align two result shapes

In `shop.db`, combine active and inactive products into columns named `item_name`, `price_amount`, and `source`. Convert cents to a real major-unit display and sort the final result.
