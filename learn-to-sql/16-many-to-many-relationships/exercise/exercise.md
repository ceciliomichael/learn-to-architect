# Exercises: Model Many-to-Many Relationships

## Exercise 1: Build and fill the relationship

1. Create `orders` and `order_items` from the lesson.
2. Insert two orders.
3. Add two different products to the first and one shared product to the second.
4. Use positive quantities and copy current prices into item prices.

## Exercise 2: Read line items and totals

1. Return one row per order item with customer and product names.
2. Add a calculated line total.
3. Return one total row per order.

## Exercise 3: Test protection

Try to insert the same product twice for one order. Explain the failure. Then find all products that have never been ordered.
