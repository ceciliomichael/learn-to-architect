# Exercises: Control Window Frames

## Exercise 1: Running total

Create a small `daily_sales` table with at least five dated rows and two equal dates. Calculate a running sum with an explicit `ROWS` frame.

## Exercise 2: Compare the default

Order by the duplicated date and compare the explicit `ROWS` result with a window sum that omits its frame. Explain peer behavior.

## Exercise 3: Moving average and full partition

Calculate a three-row moving average, then show every row beside the full-table average using an explicit unbounded frame.
