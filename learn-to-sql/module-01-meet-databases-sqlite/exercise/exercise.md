# Exercises: Meet Databases, SQL, and SQLite

## Exercise 1: Check and open SQLite

1. Check the SQLite version outside the SQLite prompt.
2. Create and enter `sql-course-practice`.
3. Open `library.db`.
4. Turn headers on and use box mode.
5. Ask `.databases` which file is open.

## Exercise 2: Create the library

1. Run the complete `CREATE TABLE books` statement from the lesson.
2. Ask `.tables` whether `books` exists.
3. Ask `.schema books` how it was defined.
4. Run the complete six-row `INSERT` from the lesson.

## Exercise 3: Read and persist

1. Select every column and row from `books`.
2. Confirm there are six visible rows.
3. Quit SQLite.
4. Reopen the same `library.db` file.
5. Select the table again to prove the database was saved.
