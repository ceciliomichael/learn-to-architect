# Exercises: Calculate Aggregate Results

## Exercise 1: Count rows and values

In `library.db`, return total books, known publication years, and distinct authors in one row.

## Exercise 2: Summarize a filtered set

Return the count, lowest price, highest price, and average price for in-stock books. Round only the displayed average to two digits.

## Exercise 3: Observe an empty set

Aggregate books with a price above 1000. Compare `COUNT(*)`, `SUM(price)`, and `AVG(price)`. Explain their results.
