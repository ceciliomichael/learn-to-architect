# Exercises: Use String and Numeric Functions

## Exercise 1: Inspect text

1. Show each title and its character length.
2. Show each author in uppercase.
3. Show the first four characters of each title.
4. Order by title length from longest to shortest, then by title.

## Exercise 2: Compose functions

Run one expression that trims the text `   database   ` and then converts it to uppercase. Explain the inner and outer result.

## Exercise 3: Present calculated numbers

1. Calculate a 7.5 percent price increase.
2. Round the result to two digits after the decimal point.
3. Name it `display_price`.
4. Explain why this is not by itself a complete plan for storing money.
