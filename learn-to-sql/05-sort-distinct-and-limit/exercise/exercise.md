# Exercises: Sort, Remove Duplicates, and Limit Results

## Exercise 1: Define order

1. List all books from newest known year to oldest, with the missing year last.
2. List books by author, then title.
3. Add `book_id` as a final tie breaker where useful.

## Exercise 2: Find unique values

1. List each author once in alphabetical order.
2. Select distinct author and stock combinations.
3. Explain why the second result can have more rows than the first.

## Exercise 3: Request a predictable portion

1. Return the two least expensive books.
2. Return the next two using `OFFSET`.
3. Explain why both queries need the same complete order.
