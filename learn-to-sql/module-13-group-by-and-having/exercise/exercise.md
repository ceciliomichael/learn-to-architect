# Exercises: Group Rows and Filter Groups

## Exercise 1: One row per author

Return each author with book count, lowest price, highest price, and rounded average price.

## Exercise 2: Filter before and after

Among in-stock books only, return authors whose average price is at least 25. Explain what `WHERE` removes and what `HAVING` removes.

## Exercise 3: Define the grain

Return one row per author and stock state with a count. Predict how many groups Lena Ortiz creates before running it.
